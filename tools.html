Tools Used

1. Rotate - The rotate tool allows you to rotate the basic shapes to face different ways and be at the angle you need. I used this to place the elements onto the rest of the image.

2. Group - The group tool is used to combine two or more shapes together to create new different or organic shapes. I used this to finalise the cutouts in the capsicum pieces.

3. Scale - The scale tool allows you to size the shapes up or down to create the right proportion for your design. I was able to scale each element so that everything was the right proportion.

4. Solid- The solid tool enables you to colour your shapes whatever colour you want, to give an aesthetic. From this I was able to use different colours in my model like red and green.

5. Hole - The hole tools allows you to add negative space to shapes, to create new shapes with cutouts. This is the initial stange for the cutouts in the capsicum.
